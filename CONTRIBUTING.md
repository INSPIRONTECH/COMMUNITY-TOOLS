# Contributing to COMMUNITY-TOOLS

Thanks for helping make NBR compliance easier for Bangladeshi SMEs.

## How to contribute

1. **Open an issue first** for any non-trivial change.
2. Fork the repo and create a feature branch: `git checkout -b fix/deadline-update`
3. Make your change, then open a PR that:
   - Links to the issue
   - Explains any NBR assumptions or sources
   - Includes a screenshot for UI changes

## Good first issues

- Fix a typo or unclear sentence in Mushak form explanations
- Add or correct one NBR deadline (include NBR source link)
- Improve a comparison table for readability
- Translate a UI label to Bangla

## Scope

Each tool lives in its own folder under the root (e.g. `TAX-CALENDAR-BD/`).
New tools follow the same pattern: self-contained, MIT-licensed, mobile-first.
