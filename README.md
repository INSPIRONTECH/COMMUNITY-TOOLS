## TAX-CALENDAR-BD

A free, self-contained web app for Bangladeshi SMEs that:

- Shows upcoming VAT (Mushak-9.1), Income Tax, TDS (Form 108), VDS, and Advance Tax deadlines
- Calculates penalty if you file late
- Exports a ready-to-import CSV for Manager.io Journal Entries
- Mobile-first, bilingual (English/Bengali), no login required

**Built for:**
- 🏪 Trading / Retail
- 🏭 Manufacturing
- 🏢 Service businesses
- 👤 Sole proprietors & individuals
- 🏦 Limited companies
- 🌐 Import / Export
